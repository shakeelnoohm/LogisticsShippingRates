# Logistics Shipping Rates

A simple Python-based shipping cost calculator that computes the total shipping cost based on package weight and rate per kilogram.

## How It Works

The script prompts the user for:
1. **Package weight** (in kilograms)
2. **Shipping rate** (per kilogram in USD)

It then calculates and displays the total shipping cost.

## Requirements

- Python 3.6+

## Usage

```bash
python Shipping_Cost_Calculator.py
```

**Example:**

```
Enter the package weight in kilograms: 2
Enter the shipping rate per kilogram: 5
Shipping Cost: 10.0 USD
```
